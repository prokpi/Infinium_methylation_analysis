# Infinium_methylation_analysis
This project is for University of Bologna MSc Bioinformatics DNA/RNA Dynamics course which aims to apply a standartisised pipeline for DNA methylation data analysis generated by the Illumina HumanMethylation450 Beadchip.

## Project Overview

This project presents a complete bioinformatics workflow for the analysis of Illumina Infinium DNA methylation array data. The analysis includes data preprocessing, quality assessment, normalization, statistical analysis, visualization, and biological interpretation of genome-wide DNA methylation profiles.

The project was implemented in **R** using Bioconductor packages commonly employed in epigenetics research.

---

## Objectives

- Import and preprocess Infinium methylation array data.
- Perform quality control and normalization.
- Explore genome-wide methylation patterns.
- Identify differentially methylated CpG sites.
- Visualize methylation profiles using publication-quality figures.
- Interpret biologically relevant methylation changes.
